# Landing Page

This repository contains a simple static landing page. To serve it via GitHub Pages:

- Ensure your repository branch `main` (or `gh-pages`) contains `index.html` at the repository root.
- GitHub Pages will serve `index.html` automatically. You can enable Pages in the repository Settings → Pages and choose the branch and root.
- A GitHub Actions workflow is included to automatically publish when you push to `main`.

Files of interest:

- `index.html` — site root (copied from `Main.html`).
- `Styles.css` — styling for the page.
- `LogoFinal copy.svg` — logo used on the page.

If you use a custom domain, add a `CNAME` file with your domain at the repository root.
